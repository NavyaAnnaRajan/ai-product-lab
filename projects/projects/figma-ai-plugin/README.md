# AI Usability Review Plugin

## Overview

Built an AI-powered Figma plugin using Claude that evaluates designs against Nielsen's 10 usability heuristics before design review.

## Why I Built It

Manual usability reviews are often time-consuming and happen late in the design process.

I wanted to explore whether AI could identify usability issues earlier and help Product Managers and Designers prioritize improvements before engineering handoff.

## Features

- Critical Issues
- Major Issues
- Easy Fixes
- Severity Dashboard
- Nielsen's 10 Heuristics
